<h1 align="center"><b>APP DE TAREFAS</b></h1>
<p>O aplicativo tem como função principal a organização e ordenação de tarefas
com uma interface minimalista e intuitiva onde a interface principal possui apenas
um input de dados onde se adiciona a tarefa e um botão para adicionar a tarefa.

Todas tarefas podem ser marcadas como concluidas, serem ordenadas em preferencia e
podem ser deletadas.

Todas tarefas são salvas em um arquivo JSON no backend para que suas tarefas estejam
salvas mesmo a pagina sendo atualizada.</p>
<br>

<h2 align="center">Tecnologias do Projeto</h2>
<p align="center">Este projeto é totalmente baseado em <b>HTML</b>, <b>CSS</b>, <b>JS</b> como frontend e <b>NodeJS</b> como backend.</p>
